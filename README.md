# threaderline

Another take on markup-centric legal document review

## Tech Stack

This project is built with:

- **Vite** - Fast build tool and development server
- **Vue 3** - Progressive JavaScript framework with Composition API
- **TypeScript** - Type-safe JavaScript
- **Vitest** - Fast unit testing framework
- **unplugin-vue-components** - Auto-import Vue components on demand
- **unplugin-auto-import** - Auto-import Vue APIs (ref, computed, etc.)
- **ESLint** - Code linting with Vue and TypeScript support
- **Prettier** - Code formatting

## Project Setup

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Run Tests

```bash
npm test
```

Run tests with UI:

```bash
npm run test:ui
```

### Lint and Format

```bash
# Run ESLint
npm run lint

# Format code with Prettier
npm run format
```

## Features

- ✅ Vue 3 with Composition API
- ✅ TypeScript support
- ✅ Auto-import for Vue APIs (ref, computed, watch, etc.)
- ✅ Auto-import for Vue components
- ✅ Vitest for unit testing with @vue/test-utils
- ✅ ESLint configured for Vue 3 and TypeScript
- ✅ Prettier for consistent code formatting
- ✅ Hot Module Replacement (HMR) for fast development
